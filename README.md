# VoxLibRus 🎙️📖

**Озвучка книг клонированным голосом автора. Русский язык. Высокое качество.**

Автор читает **1-2 страницы** (5-10 минут), VoxLibRus клонирует голос через **Qwen3-TTS** и генерирует аудиокнигу до **300 страниц** (~9+ часов).

### 🏆 Ключевые технологии

| Компонент | Модель | Качество RU | Лицензия |
|---|---|---|---|
| **ASR** (распознавание речи) | **GigaAM-v3** (Sber) | 8.4% WER | MIT |
| **TTS** (клонирование + генерация) | **Qwen3-TTS-CustomVoice** (Alibaba) | 4.5/5 | Apache-2.0 |
| TTS (опционально, макс. качество) | F5-TTS_RUSSIAN | 5/5 | CC-BY-NC |

### 🔬 Почему новый проект?

Готовых open-source решений для русской аудиокниги с voice cloning **не существует**:

- **ebook2audiobook** (19.5K⭐) — XTTS даёт 3/5 качества для RU
- **GPT-SoVITS** (59.9K⭐) — RU не primary, нет book pipeline
- **Pandrator** — Kokoro без RU, XTTS 3/5

VoxLibRus собирает **лучшие компоненты** в один пайплайн.
